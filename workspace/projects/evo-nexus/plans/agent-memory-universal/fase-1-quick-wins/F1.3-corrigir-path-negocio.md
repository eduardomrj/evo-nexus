3. Confirmar que o escopo prioritário (Clawdia, Flux, Atlas) está corrigido; aproveitar e corrigir todos os demais que apareçam no grep (Nova, Aria, Dex, Lex, Mako, Nex, Sage, Pixel, Zara, Kai, Mentor, Lumen, custom-channels) — é a mesma operação de busca/substituição, custo marginal zero.
